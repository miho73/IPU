정답 입력과 채점

이 규칙은 IPU의 채점 규칙과 방식을 설명하여 정답이 오답으로 잘못 판정되는 일이 없도록 합니다.

  1. 채점의 방식

    채점이 가능한 문제는 다음중 한 가지 방법으로 정답을 입력받습니다.

    • 직접 채점
    • 수 또는 텍스트 채점
    • 분수 채점

  2. 직접 채점

    "직접 채점"의 채점칸은 다음과 같이 생겼습니다.

    직접 채점

    정답을 입력하는 칸이 주어지지 않으며, 제출 전 자신의 답안과 문제의 답안을 비교하여 직접 정답 혹은 오답을 판단합니다. 이 종류의 문제를 채점할때는 다음과 같은 규칙에 맞게 채점해야 합니다.

    제출시 모든 직접 채점에 대해 정답/오답을 결정한 후 제출할 수 있습니다.

    • 객관적이고 양심적으로 이성에 근거하야 채점해야 합니다. 채점을 제대로 하지 않는 것은 약관 위반입니다.
    • 정답이 완전히 같지 않아도 맥락이 같고 충분히 맞다고 판단되는 경우 정답입니다.

  3. 수 또는 텍스트 채점

    "수 또는 텍스트 채점"의 정답 입력란은 다음과 같이 생겼습니다.

    수 또는 텍스트 채점

    하나의 입력 칸이 주어지며, 입력칸에 문제의 정답을 적습니다. 이때 문제에 특별한 지시가 없는 한 다음과 같이 정답을 적습니다.

    • 가짓수, 경우의 수, 개수, 각도 등은 단위를 적지 않습니다.
    • 길이, 넓이, 부피, 압력, 속도 등 단위를 붙여야 하는 경우 문제에서 정한 단위를 사용하고 답에는 단위를 적지 않습니다. 만약 문제에서 단위를 정하지 않은 경우 풀이중 단위의 접두어를 바꾸지 않습니다.
    • 양수를 뜻하는 +기호는 반드시 생략합니다.
    • 소수점은 '.'으로 적습니다.
    • 정답 외의 문자(정답 앞뒤의 공백 등)은 적지 않습니다.

  4. 분수 채점

    "분수 채점"의 정답 입력란은 다음과 같이 생겼습니다.

    분수 채점

    두 개의 입력 칸이 주어지며, 첫번째 입력 칸에는 문제의 정답 중 분자 부분을, 두번째 입력 칸에는 분모 부분을 적습니다. 이때 문제에 특별한 지시가 없는 한 다음과 같이 정답을 적습니다.

    • 분자와 분모 둘 모두 단위를 적지 않습니다.
    • 분자와 분모는 정수로 적습니다.
    • 정답은 기약분수로 약분되어 있어야 합니다.
    • 정답이 음의 유리수인 경우 부호는 분자에 적습니다.
    • 정답 외의 문자(정답 앞뒤의 공백 등)은 적지 않습니다.

  5. 예시

    다음은 정답과 정답칸에 적는 정답의 예시입니다.

    문제 정답 써야 하는 정답
    6이하인 자연수의 개수를 구하시오. 주사위를 던져서 나올 수 있는 눈의 경우를 구하시오. 6개 6
    1A의 전류가 5Ω인 저항의 양단의 전위차(V)를 구하시오. 10V의 전원에 연결된 크기가 같은 두 저항 중 하나에 걸리는 전압을 구하시오. 5V 5
    AB의 길이가 3cm, BC의 길이가 3cm인 정삼각형 ABC의 한 변 CA의 길이(cm)를 구하시오. 3cm 3
    AB의 길이가 3, BC의 길이가 3인 정삼각형 ABC의 한 변 CA의 길이를 구하시오 3 3
    5보다 10 작은 수를 구하시오. -5 -5
    동전을 던져 앞면이 나올 확률을 구하시오. $\frac{1}{2}$ 분자: "1", 분모: "2"
    기울기가 4인 직선과 수직한 직선의 기울기를 구하시오. $-\frac{1}{4}$ 분자: "-1", 분모: "4"
